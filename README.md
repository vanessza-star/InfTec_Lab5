# Лабораторна робота №4 з дисципліни "Інформаційні технології управління"

## Тема
Робота з гілками, комітами та Pull Request у GitHub.

## Мета
Ознайомитись з основними командами Git:
- створення та злиття гілок;
- створення комітів;
- створення Pull Request;
- виконання rebase.

## Виконання роботи
1. Створено дві гілки: feature_Task9 та feature_Task10.
2. Злиття гілок виконано у main.
3. Створено Issue про відсутність README файлу.
4. Створено коміт, який вирішує Issue (додано цей файл).
5. Виконано rebase.
